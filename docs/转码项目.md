# 转码项目

转码项目分为完全0基础（完全没有修过cs先修课程）和部分0基础转码项目（修过部分cs先修课程），如果一个同学从经济/商科/生物等系，且没有修过诸如数据结构、计算机组成原理、操作系统等先修课，
可以参考[转码同学需要补的专业课](useful_docs/转码需要补的专业课)来进行补课
